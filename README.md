Prerequisites
-------------

- [MongoDB](https://www.mongodb.org/downloads)
- [Node.js 6.0+](http://nodejs.org)
- Command Line Tools
 
Starting :
    chmod -x script.sh
    ./script.sh
    sudo mongod

  in a another Terminal in Sneakers4Lumberjacks''folder :
    node app.js

Then :

Go on http://localhost:8080
